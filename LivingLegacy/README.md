# LivingLegacy.commercecult.app

This project hosts the Living Legacy media kit and awareness campaign integrated with CommerceCult.

## Features
- Complete Media Kit for Living Legacy end-of-life concierge services
- Next.js frontend deployed on Vercel
- GitHub Actions for automated deployment
- Agentic Bot for awareness campaigns and social media automation
